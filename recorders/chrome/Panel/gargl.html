You can specify parameterized values by inserting @some-parameter-name@ within the URL. This will specify to gargl generators that this gargl function takes a parameter called some-parameter-name that will become a part of the url of the request. The URL can have one or more parameterized values defined.
Example: "www.somesite.com/@category@/@topic@/" would create a url with category and topic parameters.
